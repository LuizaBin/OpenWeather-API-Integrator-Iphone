# OpenWeather-API-Integrator-Iphone
An iOS implementation of the OpenWeatherMap REST API, developed to showcase practical skills in secure endpoint connectivity, JSON response parsing, and real-time data formatting

Implementation Overview
As an individual with previous experience working with APIs, I developed this shortcut to demonstrate how professional-grade data can be integrated into the iOS ecosystem.

**Key Technical Highlights:**

REST API Consumption: Successfully integrated the OpenWeatherMap API to fetch live meteorological data.

Structured Data Handling: Navigated complex JSON dictionaries to extract specific keys such as main.temp and main.feels_like

Parameter Management: Configured API call parameters to handle unit conversion (Kelvin to Metric) and location-based queries.

Security Standards: Utilized HTTPS for all network requests to ensure data privacy and integrity during transmission.

**Key Features**

Secure API Integration: Uses https to protect data in transit.

Data Parsing: Extracts temp, humidity, and weather.description from nested JSON.

User Experience: Includes logic to round decimal temperatures and format them into natural language.

**Setup Instructions**
1. Download the .shortcut file and open it on an iOS device.

2. Get a free API key from OpenWeatherMap.

Paste your key into the designated Text field at the top of the shortcut.

Run the shortcut and enter any city name.
